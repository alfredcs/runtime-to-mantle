# Environment Configuration

Before you run a notebook, walk through the four sub-pages below in order.
Each one is a ~3-minute click-through that leaves you with exactly what
the next one needs.

| # | Step | What you'll do |
|---|---|---|
| 1 | **Workshop Access** | Join the event and log in to the AWS console |
| 2 | **IAM Permissions** | Attach the Mantle + SageMaker policies to your role |
| 3 | **Amazon Bedrock Mantle Access** | Enable model access for the four models used in the labs |
| 4 | **Amazon SageMaker AI Notebook** | Open JupyterLab in the pre-provisioned notebook instance |
| 5 | **Clone the Repository** | Pull the workshop code from GitHub |

> **Region:** Every step in this workshop runs in **`us-east-1`**. Mantle
> is region-scoped; Bearer tokens minted in one region will be rejected
> by the endpoint of another. See each sub-page for the exact region
> toggles.

When all five steps are green, head to **Lab 1** in the navigation.
