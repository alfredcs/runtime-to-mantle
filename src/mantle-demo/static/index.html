Keep bedrock-runtime when…

  • You need Amazon Nova / Titan, Llama, Cohere (not on Mantle)
  • Batch inference, bidirectional speech streaming, or GovCloud
  • Native Guardrails / Knowledge Bases / Agents / Flows integration
The honest answer for most enterprises is a hybrid: route Mantle-compatible traffic to Mantle behind an abstraction, keep runtime for the rest.
